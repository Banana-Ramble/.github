We make STEM you can hold — apps and real hardware kits where kids flip actual switches, run honest physics, and learn by doing. **Play first. Lessons follow.**

## Why Banana Ramble

- 🛠️ **Learning you can hold** — Real controls in real hands. The fastest way to understand a system is to run it yourself.
- ⚙️ **Honest under the hood** — No scripted outcomes. Genuine physics and logic mean the world reacts to what a kid actually does.
- 🎈 **Play first** — If it isn't fun, it doesn't teach. Every product earns its lesson through delight.
- 🧰 **Built to last** — Sturdy hardware, open build files, and software we keep improving long after launch.

## How we make STEM stick

**Build it. Run it for real. Learn by playing.** Every Banana Ramble product shares one loop — and that loop is where the learning hides.

1. **Build it** — Snap a kit together: a control panel, a cardboard rocket, a word list. Making it is step one of understanding it.
2. **Run it for real** — Switches, dials, and gauges wired to a live simulation. Cause and effect you can feel, not a canned animation.
3. **Learn by playing** — Progress, scenarios, and mastery challenges turn practice into a game worth finishing.
